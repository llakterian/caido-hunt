   2. Find icon: "Caido Hunt Scanner" 🛡️
   3. Double-click it
   4. ✨ GUI opens automatically!

┗━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━┛

   1. Press Super/Windows key
   2. Type "Caido Hunt"
   3. Click the application
   4. ✨ GUI opens automatically!

┗━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━┛

   cd ~/Desktop/caido-hunt
   ./start.sh

┌────────────────────────────────────────────────────────────────────────┐
│  🎨 WHAT HAPPENS WHEN YOU CLICK                                        │
└────────────────────────────────────────────────────────────────────────┘

   ✅ Terminal opens
   ✅ Virtual environment activates (caido-env)
   ✅ Dependencies checked/installed automatically
   ✅ GUI starts on http://127.0.0.1:5000
   ✅ Browser opens automatically
   ✅ READY TO SCAN! 🎯

┌────────────────────────────────────────────────────────────────────────┐
│  📝 USING THE SCANNER                                                  │
└────────────────────────────────────────────────────────────────────────┘

   1. Enter target URL (e.g., https://testphp.vulnweb.com)
   2. Set scan options:
      • Threads: 10
      • Delay: 1.0s
      • Timeout: 30s
   3. Click "Start Scan"
   4. Watch vulnerabilities appear in real-time
   5. Export results as JSON or CSV

┌────────────────────────────────────────────────────────────────────────┐
│  ⚠️  IMPORTANT: SCAN RESPONSIBLY!                                      │
└────────────────────────────────────────────────────────────────────────┘

   ✅ ONLY scan targets you have permission to test
   ✅ Get written authorization before scanning
   ✅ Follow bug bounty program rules
   
   ❌ DO NOT scan without permission
   ❌ DO NOT exceed authorized scope

┌────────────────────────────────────────────────────────────────────────┐
│  🔧 TROUBLESHOOTING                                                    │
└────────────────────────────────────────────────────────────────────────┘

   If icon doesn't work:
   
   1. Open terminal:
      cd ~/Desktop/caido-hunt
      ./launch_caido_hunt.sh
   
   2. Check virtual environment:
      ls -la caido-env/
   
   3. Manual activation:
      source caido-env/bin/activate
      python simple_gui.py
