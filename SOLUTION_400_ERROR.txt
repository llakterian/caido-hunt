The 400 error happens when:
   1. Port 5000 is already in use
   2. Old GUI instance is still running
   3. Multiple GUIs started at once

TO PREVENT:
   ✅ Always close previous GUI before starting new one (Ctrl+C)
   ✅ Check if GUI is running: lsof -i:5000
   ✅ Use different ports for multiple instances
   ✅ Close browser tab when done

TO FIX QUICKLY:
   Run this command to clear port 5000:
   
   lsof -ti:5000 | xargs kill -9

   Or use the diagnostic script:
   ./diagnose.sh
