┌────────────────────────────────────────────────────────────────────────┐
│  🚀 HOW TO USE                                                         │
└────────────────────────────────────────────────────────────────────────┘

METHOD 1: Desktop Icon
   • Double-click "Caido Hunt Scanner" on desktop
   • New real-time GUI opens automatically!

METHOD 2: Terminal
   cd ~/Desktop/caido-hunt
   ./start.sh

   Or directly:
   python realtime_gui.py --port 5000

TOP SECTION: Configuration
   • Target URL input

   • Start/Export buttons

MIDDLE SECTION: Live Statistics
   • Elapsed Time - 0s (updates every second!)
   • Pages Scanned - Real count
   • Total Requests - Actual number
   • Vulnerabilities - Found in real-time

MAIN SECTION: Live Terminal
   • Dark theme console
   • Real-time log output
   • Color-coded messages
   • Auto-scrolling
   • Shows EXACTLY what the scanner is doing

BOTTOM SECTION: Vulnerability Breakdown
   • Only appears when vulnerabilities found
   • Shows Critical/High/Medium/Low counts
   • Color-coded badges

Technology:
   • Flask web server
   • Server-Sent Events (SSE) for real-time streaming

   • Bootstrap 5 + Font Awesome icons
   • Real-time statistics updates every second

Port:
   • Default: 5000

Auto-features:
   • Auto-opens browser
   • Auto-scrolls terminal output
   • Auto-updates statistics
   • Auto-highlights vulnerabilities
